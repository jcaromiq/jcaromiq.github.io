Vitaminando la programacion funcional de java 8 con Vavr

2017-08-21 200 words 1 minutes

Con la release de java 8, se abrió un nuevo mundo en java para desarrollar “funcionalmente”. Pero ¿es suficiente con lo que trae java 8 de serie?

Vavr tal y como se describen, es una librería funcional para java 8+ que ayuda a reducir el código y añadirle robustez. Uno de los principios de la programación funcional es que los datos sean inmutables, para ello, entre otras cosas, vavr incluye colecciones inmutables y las necesarias funciones para operar con ellas.

¿Que tiene vavr que ha hecho que sea una librería a seguir muy de cerca? lo primero es la documentación que tienen en su web muy detallada.

También que es open source y tiene todo su código publicado en Github

Una vez dentro de la librería, tiene cosas tan útiles como los Eithers, Futures, Validations

En próximos posts detallaré con ejemplos cada una de las funcionalidades que más uso.
